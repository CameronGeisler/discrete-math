Theorem 8.3.7.

Let \(X, Y\) be finite sets, \(f: X \rightarrow Y\text{.}\) Then,

  • \(f\) is injective if and only if \(\abs{X} \leq \abs{Y}\text{.}\)

  • \(f\) is surjective if and only if \(\abs{X} \geq \abs{Y}\text{.}\)

  • \(f\) is bijective if and only if \(\abs{X} = \abs{Y}\text{.}\)
