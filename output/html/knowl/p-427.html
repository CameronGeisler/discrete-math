By induction, for \(n = 1\text{,}\) we have \(1 \cdot (1 + 1) = 2\) which is even. Then, assume the result is true for some \(n \in \mathbb{Z}, n \geq 1\text{,}\) that \(n(n + 1)\) is even. Then, we want to show that \((n + 1)((n + 1) + 1) = (n + 1)(n + 2)\) is even. Then,

\begin{align*} (n + 1)(n + 2) \amp = n^2 + 3n + 2\\ \amp = (n^2 + n) + (2n + 2) \amp\amp \text{splitting up the terms to use the induction hypothesis}\\ \amp = n(n + 1) + 2(n + 1) \end{align*}

By the induction hypothesis, \(n(n + 1)\) is even, and also \(2(n + 1)\) is even, so their sum \((n + 1)(n + 2)\) is even.
