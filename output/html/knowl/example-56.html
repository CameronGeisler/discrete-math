Example 7.1.7. Empty set is a subset of every set.

The empty set is a subset of all other sets. In other words, for all \(A\text{,}\) \(\emptyset \subseteq A\text{.}\)

The empty set is a strict subset of every set, except for the empty set.

This is because \(\emptyset \subseteq A\) means that for all \(a \in \emptyset\text{,}\) \(a \in A\text{.}\) This statement is vacuously true, because “all” of the zero elements in \(\emptyset\) are indeed in \(A\text{.}\)
