Proof

By induction, for \(n = 1\text{,}\) we have,

\begin{equation*} \sum_{i=1}^1 i^3 = 1^3 = 1 = \brac{\frac{1(1 + 1)}{2}}^2 \end{equation*}

Then, assume that the result is true for some \(n = k \in \mathbb{N}\text{.}\) Then, we want to show that,

\begin{align*} 1^3 + 2^3 + \dots + (n - 1)^3 + n^3 + (n + 1)^3 = \sum_{i=1}^{n+1} i^3 \amp = \brac{\frac{(n + 1)(n + 2)}{2}}^2\\ \amp = \frac{(n + 1)^2(n + 2)^2}{4} \end{align*}

Then,

\begin{align*} 1^3 + 2^3 + 3^3 + \dots + (n - 1)^3 + n^3 + (n + 1)^3 \amp = \sum_{i=1}^{n+1} i^3\\ \amp = \sum_{i=1}^n i^3 + (n + 1)^3 \amp\amp \text{separating off the final term}\\ \amp = \brac{\frac{n(n + 1)}{2}}^2 + (n + 1)^3 \amp\amp \text{by the induction hypothesis}\\ \amp = \frac{n^2(n + 1)^2}{4} + (n + 1)^3\\ \amp = \frac{n^2(n + 1)^2 + 4(n + 1)^3}{4}\\ \amp = \frac{(n + 1)^2 (n^2 + 4n + 4)}{4} \amp\amp \text{factoring $(n + 1)^2$ from both terms}\\ \amp = \frac{(n + 1)^2(n + 2)^2}{4} \end{align*}
