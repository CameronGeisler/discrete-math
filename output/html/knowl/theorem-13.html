Theorem 4.2.4.

  1. Sum of 1s.

    \begin{equation*} \sum_{i=1}^n 1 = \underbrace{1 + 1 + \dots + 1}_{\text{$n$ terms}} = n \end{equation*}

  2. Sum of the first n natural numbers.

    \begin{equation*} \sum_{i=1}^n i = 1 + 2 + 3 + \dots + n = \frac{n(n + 1)}{2} \end{equation*}

  3. Sum of the first n squares of natural numbers.

    \begin{equation*} \sum_{i=1}^n i^2 = 1^2 + 2^2 + 3^2 + \dots + n^2 = \frac{n(n + 1)(2n + 1)}{6} \end{equation*}

  4. Sum of finite geometric series. For \(r \neq 1\text{,}\)

    \begin{equation*} \sum_{i=1}^n r^{i-1} = 1 + r + r^2 + \dots + r^{n-1} = \frac{r^n - 1}{r - 1} \end{equation*}
