By induction, for \(n = 1\text{,}\) we have,

\begin{equation*} \sum_{i=1}^1 (2i - 1) = 1 = 1^2 \end{equation*}

Then, assume the result is true for some \(n = k \in \mathbb{N}\text{,}\) and we want to show that,

\begin{equation*} 1 + 3 + \dots + (2k - 1) + (2(k + 1) + 1) = \sum_{i=1}^{k+1} (2i - 1) = (k + 1)^2 \end{equation*}

Here, note that the next odd number in the summation is \(2(k + 1) - 1 = 2k + 1\text{.}\) Then,

\begin{align*} 1 + 3 + \dots + (2k - 1) + (2k + 1) \amp = \sum_{i=1}^{k+1} (2i - 1)\\ \amp = \sum_{i=1}^k (2i - 1) + (2k + 1) \amp\amp \text{separating off the final term}\\ \amp = k^2 + (2k + 1) \amp\amp \text{by the induction hypothesis}\\ \amp = (k + 1)^2 \amp\amp \text{by factoring} \end{align*}

Thus, by induction, the result is true for all \(n \in \mathbb{N}\text{.}\)
