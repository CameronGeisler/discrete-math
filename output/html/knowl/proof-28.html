Proof

Let \(n\) be the largest positive integer. Then, \(n\) is a positive integer, so \(n \geq 1\text{.}\) Then, multiplying both sides by \(n\) (which is positive), we have \(n^2 \geq n\text{.}\) Also, \(n\) is a positive integer so \(n^2\) is also a positive integer, so it must be at most \(n\text{,}\) or \(n^2 \leq n\text{.}\) Thus, \(n^2 = n\text{,}\) so \(n^2 - n = 0\text{,}\) so \(n(n - 1) = 0\text{.}\) The solutions to this equation are \(n = 0, 1\text{.}\) Since 0 is not a positive integer, this implies that \(n = 1\text{.}\)
