Theorem 3.1.2. Fundamental counting principle.

Let there be \(k\) choices, with \(n_1\) ways to make choice 1, \(n_1\) ways to make choice \(2, \dots, n_k\) ways to make choice \(k\) (in general for choice \(i\text{,}\) there are \(n_i\) ways). Then, there are,

\begin{equation*} n_1 n_2 \cdots n_k \end{equation*}

ways to make the entire sequence of choices. More precisely, if there are \(n_1\) distinct objects in set 1, \(n_1\) in set \(2, \dots, n_k\) in set \(k\text{,}\) then there are \(n_1 n_2 \cdots n_k\) ways to choose one object from each set.
