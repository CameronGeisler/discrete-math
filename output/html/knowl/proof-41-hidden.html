By induction, for \(n = 1\text{,}\) we have,

\begin{equation*} \sum_{i=1}^1 i = 1 = \frac{1(1 + 1)}{2} \end{equation*}

Then, assume the result is true for some \(n = k \in \mathbb{N}\text{,}\) and we want to show that,

\begin{equation*} \sum_{i=1}^{k+1} = \frac{(k + 1)((k + 1) + 1)}{2} = \frac{(k + 1)(k + 2)}{2} \end{equation*}

Then,

\begin{align*} 1 + 2 + \dots + (k - 1) + k + (k + 1) \amp = \frac{k(k + 1)}{2} + (k + 1) \amp\amp \text{by the induction hypothesis}\\ \amp = \frac{k(k + 1)}{2} + \frac{2(k + 1)}{2}\\ \amp = \frac{k^2 + 3k + 2}{2} \amp\amp \text{combining fractions and simplifying}\\ \amp = \frac{(k + 1)(k + 2)}{2} \amp\amp \text{factoring the quadratic expression} \end{align*}

By induction, the result is true for all \(n \in \mathbb{N}\text{.}\)
